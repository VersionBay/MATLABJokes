joke

display a joke

Syntax

joke()
joke(jokeNumber=1)
joke(languague="english")
joke(jokeNumber=2, languague="english")
joke(languagu="portuguese",jokeNumber=1)

Description

This function displays a joke in the command window.

Examples

joke()
joke(jokeNumber=1)
joke(languague="english")
joke(jokeNumber=2, languague="english")
joke(languagu="portuguese",jokeNumber=1)

Input Arguments

  • jokeNumber - specify which joke you want to display, when omited a random joke is choosen
  • languague - specify which language is used. Default is "english". Options are: "english", "portuguese", "dutch"

Output Arguments

None
